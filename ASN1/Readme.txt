To compile emma-asn1.c, run the command gcc emma-asn1.c -o EMMA-ASN1

This should create an executable file named EMMA-ASN1. 

Proceed by running the command ./EMMA-ASN1

This should run the program starting with a prompt to enter a sentence. Clicking enter will display 

each word on a new line and save the output to a text file called Output.txt.

To view the text file, run the command nano Output.txt and hit enter.
